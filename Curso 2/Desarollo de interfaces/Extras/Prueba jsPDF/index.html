Para crear un fichero pdf desde nuestra aplicación HTML-JS-CSS,
vamos a utilizar la librería jsPDF de la api npm.

Para utilizar esta librería vamos a utilizar el fichero jspdf.min.js
e importar la clase jsPDF, o utilizar el link para poder trabajar CDN
desde la página de npm.
Vamos a trabajar con la versión 1.3.5, versiones recientes resultan problemáticas de importar de esta manera.
